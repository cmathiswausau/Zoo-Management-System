# Zoo Management System

A C# object-oriented application that models core zoo operations, including animals, habitats, and related behaviors.

## 📌 Overview

This project was developed using instructor-provided requirements and structure to apply object-oriented programming concepts in a structured scenario. The implementation focuses on designing class relationships and building logic that reflects a functioning system.

## 🎯 Key Features

* Representation of animals and habitats
* Structured class relationships and interactions
* Encapsulation using properties and methods
* Organized multi-class architecture

## 🧠 Concepts Demonstrated

* Object-oriented design (classes, relationships, abstraction)
* Encapsulation and controlled data access
* Debugging and execution tracing
* Logical problem-solving within defined requirements
